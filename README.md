# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript. This project showcases my skills, projects, education, and contact information.

## ✨ Features

- Responsive Design
- About Me Section
- Skills Section
- Projects Showcase
- Resume Download
- Contact Section
- Social Media Links

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
